Instrucciones:
- Descomprime el ZIP.
- Pegá tus imágenes dentro de la carpeta 'img/' con nombres como 1.jpg, 2.jpg, etc.
- Añade las etiquetas <img src="img/1.jpg" alt="Descripción"> dentro del div#gallery en index.html.
- Sube la carpeta completa a tu hosting o GitHub Pages.
- Usa el botón superior derecho para cambiar entre modo claro y oscuro.
